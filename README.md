# 🤖 Event Management Chatbot

A Python-based command-line chatbot that streamlines event planning by collecting client information, storing event details, tracking tasks, and sending confirmations and reminders — all through a conversational interface.

---

## 💡 Project Overview

Managing events involves juggling multiple clients, venues, dates, and tasks simultaneously. This chatbot simplifies that process by acting as an interactive assistant that guides users through the full event management workflow — from onboarding a new client to tracking every task associated with their event.

Built entirely in Python as a rule-based conversational system, it runs directly in the terminal with no external dependencies or internet connection required.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 👤 Client Data Collection | Collects and stores client name, contact details, and preferences |
| 📅 Event Details Storage | Records event type, date, venue, and key requirements |
| ✅ Task Tracking | Maintains a to-do list of all tasks associated with each event |
| 🔔 Confirmations & Reminders | Sends confirmation messages and reminds users of upcoming tasks |
| 💬 Conversational Interface | Rule-based dialogue that guides users step by step |

---

## 🛠️ Tools & Technologies

- **Language:** Python 3
- **Interface:** Command-line (terminal)
- **Logic:** Rule-based scripted responses
- **Data Storage:** In-memory Python data structures (dictionaries, lists)

---

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/vighnesh-tn/chatbot.git
cd chatbot
```

2. Run the chatbot:
```bash
python chatbot.py
```

3. Follow the prompts in the terminal to:
   - Add a new client
   - Enter event details
   - Track tasks
   - View confirmations and reminders

## 📌 Example Interaction

```
Bot: Welcome to Event Manager! What's your name?
User: John Smith

Bot: Hi John! What type of event are you planning?
User: Wedding

Bot: Great! What is the event date?
User: June 15, 2025

Bot: What venue have you selected?
User: Grand Ballroom, Dallas TX

Bot: Got it! I've saved your event details. Let's now add tasks for your event.
Bot: Task 1 added: Book catering ✅
Bot: Reminder set for June 1, 2025 — 2 weeks before your event!
```
